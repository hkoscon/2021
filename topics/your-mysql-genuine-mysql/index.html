Is your MySQL the genuine MySQL

event 2021-07-17 (Day 1) ~ 14:35 - 15:05
explore Room 1
  • comment English
  • network_check Beginner

Are you sure your MySQL is actually MySQL? Not any of imitation or fake one? This session will cover topics from which are official download sites and repo of MySQL, which is the genuine MySQL image for Docker, how to verify downloaded files are real ones, and which is the DBaaS based on genuine MySQL.

Ryusuke Kajiyama

Oracle Japan / Japan

Ryusuke Kajiyama ( 梶山隆輔 ) is Open Source Solution Specialist with 15+ years of experience of systems design using MySQL and open source solutions. He originally joined MySQL AB, and is currently managing MySQL Solution Engineering team of Asia Pacific region at Oracle.
